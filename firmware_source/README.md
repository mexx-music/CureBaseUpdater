# firmware_source

Drop the firmware repository or firmware-related source files here for reference. This folder is intended as a copy area only — do not expect builds to run from here unless you also add the proper project files.

Usage:
- Copy your firmware sources into this folder (via Finder or `cp -r /path/to/firmware/* firmware_source/`).
- Keep the original repository layout inside this folder if you need to run firmware-specific tools.
